Advantages

Why Trainium?

Purpose-built silicon with a full-stack software approach enables developers to ship faster at lower cost.

⚡

High Performance, Low Cost

Trn3 UltraServers deliver 362 MXFP8 PFLOPs with 4× better energy efficiency than Trn2. Up to 50% lower training costs on Trn1 vs comparable GPU instances.

🔧

Native Framework Support

AWS Neuron integrates natively with PyTorch, JAX, Hugging Face, vLLM, and PyTorch Lightning. No code changes required for most workloads.

🧮

State-of-the-Art Precisions

Full support for BF16, FP16, FP8, MXFP8, and MXFP4 mixed-precision training. Hardware-accelerated stochastic rounding and micro-scaling for accuracy.

🔗

Scale-Up Interconnect

NeuronLink and NeuronSwitch-v1 enable ultra-low-latency all-to-all collectives across 144 chips in a single UltraServer, delivering 706 TB/s memory bandwidth.

🔍

Deep Developer Access

Neuron Kernel Interface (NKI) exposes the full Trainium ISA for instruction-level programming. Neuron Explore provides full stack visibility from code to hardware engines.

☁️

AWS Ecosystem Integration

Works seamlessly with SageMaker, SageMaker HyperPod, EKS, ECS, ParallelCluster, AWS Batch, and third-party tools like Ray, Domino Data Lab, and Datadog.

train.py 
# Minimal change from standard PyTorch
import torch
import torch_xla.core.xla_model as xm

# Get Trainium device (replaces "cuda")
device = xm.xla_device()

model = MyTransformerModel().to(device)
optimizer = torch.optim.AdamW(
    model.parameters()
)

for batch in dataloader:
    x = batch[0].to(device)
    loss = model(x).loss
    loss.backward()
    xm.optimizer_step(optimizer)
    xm.mark_step()  # triggers XLA compile

Evolution

The Trainium Arc

From first silicon to 3nm frontier AI accelerators — an unbroken line of purpose-built innovation.

2021 — Announced · 2022 — GA
Trainium1 · Trn1 Launch

AWS's first custom ML training chip enters general availability. Trn1.32xlarge features 16 NeuronCore v2 chips, 512 GB HBM, and 800 Gbps EFA networking. Delivers up to 50% lower training costs than comparable GPU instances. Customers including Ricoh, Karakuri, and Arcee AI adopt early.

NeuronCore v2 32 GB HBM / chip BF16 · FP16 · FP8 Neuron SDK v2
2022 — Launch
trn1n.32xlarge — High-Bandwidth Variant

The trn1n variant doubles EFA bandwidth to 1,600 Gbps via 16 × 100 Gbps NIC interfaces, optimizing multi-node distributed training across large clusters.

1,600 Gbps EFA 16 × 100 Gbps NICs Multi-node scale-out
2023–2024
Trainium2 · Trn2 & UltraServers

NeuronCore v3 delivers 4× the performance of Trainium1. Trn2 UltraServers scale to 64 chips with NeuronLink proprietary interconnect, enabling training of 1T+ parameter models. 30–40% better price performance than GPU-based P5e and P5en instances. Anthropic, Databricks, and poolside among early adopters.

NeuronCore v3 4× vs Trn1 64-chip UltraServer MXFP8 4× Sparsity (16:4)
2025 — re:Invent Announced & GA
Trainium3 · 3nm · NeuronCore v4

AWS's first 3nm AI chip — purpose-built for agentic, reasoning, and video generation workloads. 2× higher compute performance at 2.52 PFLOPs FP8 per chip, 1.5× more memory (144 GB HBM3e), 1.7× memory bandwidth (4.9 TB/s). Trn3 UltraServers scale to 144 chips with NeuronSwitch-v1, delivering 362 MXFP8 PFLOPs and 4× better energy efficiency vs Trn2 UltraServers.

3nm process NeuronCore v4 144-chip UltraServer MXFP8 · MXFP4 NeuronSwitch-v1 4× energy efficiency

AWS deepens developer access with Neuron Kernel Interface (NKI), exposing the full Trainium ISA for instruction-level kernel development. Open-sourced Neuron Kernel Library provides ready-to-deploy optimized kernels. Neuron Explore connects developer code to hardware-level engine visibility.
